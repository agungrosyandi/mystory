UI/UX Web Design

To build Interactive and functional Website, you need a specifics tools. For flexibility and easy to collaborate with the other, Figma is the best choice. First steps, create most important aspect in Web Development, that's structure and visual design concept, including wireframe, pattern, and navigate structure, for example click here . After it finishes in the design concept, then create and develop the programming code. In general, HTML, CSS and Javascript is commonly used for the Web Development to create Static Web or Landing Page and Dynamic Web or Web App. For Landing Page, pure HTML, CSS, Javascript and GSAP library is more than enough to create the interactive and beautiful website. However for more complex project like an Web App, you need Javascript framework such a React/NextJS for flexibility, clean component, and easy to maintance or update. Lastly after finalizes the project, both include the design concept and code programming, push the workflow using github, then choose hosting and domain to deploy the finished project.
